# aoc21.adamkiss.com

> Advent of Code 2021 - Solutions as Kirby site

---

## Installation

``` bash
composer install -no
npm run ci

# Develop front-end
npm run start
# Bundle assets for production
npm run prod
```

## Unstable (-alpha/-rc) dependencies

- TailwindCSS is installed as 3.0.0-alpha2
